{sever_url}/image/[id]


You can create your own game from this url

{server_url}/creategame?word=[word]

Which will return an id for others to play the game


You can create a game, with or without an id, if no id is provided you will get a random Icelandic word

{server_url}/newgame?game_id=[game_id]
